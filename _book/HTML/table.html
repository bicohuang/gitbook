第一节

table 表格笔记

1、table 标签

  1. name 属性  定义名称
  2. border属性   表示table边框的粗细， 默认是1
  3. cellpadding属性 表示格子和格子之间的距离
  4. cellspacing属性 表示格子与内容的距离
  5. align属性  定义内容的对齐方式
    • left  左对齐
    • center  中心对齐
    • right  右对齐
  6. valign属性 垂直排列方式
    • top  顶部对齐
    • middle  中间对齐
    • bottom  底部对齐

4、a标签

定义页面跳转

  1. href属性 表示要跳转过去的地方
  2. title属性 表示链接栏的提示信息
  3. target属性 定义打开新页面的方式
    • _blank  新的空白页
    • _self  当前页
    • _top  顶上

锚点

表示跳转的地方在同一页面内,href属性写的是该地方的id

5、img标签

展示图片

  1. src  表示图片的出处
    • 相对地址 在项目文件里，相对HTML的位置
      1. ./ 表示当前目录
      2. ../ 返回上层目录
      3. image/ 进入一层目录
      4. ../image/ 先返回上层，再进入image目录
    • 绝对地址 不在项目文件里，例如引用网上的图片
  2. alt  定义图片的替代文本
  3. width 设置图片的宽度
  4. height 设置图片的高度

图片热区

  1. Map 表示热区集合
    1. area 表示单个热区
      • shape  热区形状
      • coords  形状的坐标值
<img src="https://www.baidu.com/img/bd_logo1.png" usemap="map"/>
<map name="map">
<area shape="rect" coords="0,0,400,400" href="fisrt.html" alt="加载失败" />

第三节 工具のgulp

1、gulp的安装与使用

1、gulp的安装

  1. 进行gulp的下载
    • npm install cnpm -g --registry=https://registry.npm.taobao.org
      gulp 的淘宝镜像链接
  2. 安装gulp
    • cnpm install gulp -g
      gulp 的全局安装

  3. 加载gulp

    • cnpm install gulp --save -dev

    • cnpm install
      gulp 的局部安装

  4. 运行gulp
    • gulp dev
      gulp 的开发环境
    • gulp pro
      gulp 的生产环境

备注：在项目的当前路径运行 cmd

2、gulp的使用

  • 运行 gulp
  • 每次写完页面后，再次运行 gulp dev
  • 每个项目都要再次安装，加载

    新建项目要创建src  里面创建css  font  html  images  is  media  scss七个小文件
